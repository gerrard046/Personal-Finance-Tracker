# Icon for Personal Finance Tracker App

This directory should contain:
- icon.png (512x512 or larger - for app icon)
- icon.ico (for Windows executable)

The icon will be automatically converted by electron-builder.
If you don't have an icon, the app will use a default Electron icon.

To add a custom icon:
1. Create a 512x512 PNG image named "icon.png"
2. electron-builder will automatically convert it to .ico format
3. Place the icon.png file in this directory
